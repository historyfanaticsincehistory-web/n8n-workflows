# n8n Workflows

Automated workflow building met AI agents.

## Hoe werkt het

1. Maak een issue met label `build`
2. Beschrijf wat je wilt
3. 's Nachts bouwen Codex + Claude de workflow
4. 's Ochtends keur je de PR goed

## Structuur

```
/workflows    - n8n workflow JSON bestanden
/tests        - Test cases
/.github      - GitHub Actions
```

## Issues aanmaken

Gebruik het "Build Request" template.
